QUICK REFERENCE
===============

Setup:
  1. cp .env.example .env
  2. Edit .env with your paths/hosts
  3. mkdir -p /models && chmod 777 /models

Start All Models:
  Blackwell:  ./scripts/start-model.sh llama33 && ./scripts/start-model.sh qwen36
  A30:        ./scripts/start-model.sh mimo && ./scripts/start-model.sh llava

View Status:
  ./scripts/status.sh

Test Models:
  ./scripts/test-model.sh llama33
  ./scripts/test-model.sh --all

Query via LiteLLM:
  curl -X POST http://localhost:4000/v1/completions \
    -H "Content-Type: application/json" \
    -d '{"model":"llama33","prompt":"Hello","max_tokens":50}'

Switch Models (MIMO ↔ DeepSeek):
  Enable DeepSeek:   ./scripts/swap-deepseek.sh --enable
  Disable DeepSeek:  ./scripts/swap-deepseek.sh --disable
